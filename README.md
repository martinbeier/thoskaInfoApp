# thoskaInfoApp

View your remaining balance from thoska using your NFC-equipped Android phone!
(for thoska based on MF Classic the correct key is required)

## Supported Protocols

* [MIFARE DESFire][3]
* [MIFARE Classic][4]

## Supported Phones

thoskaInfoApp requires an NFC Android phone running 4.1 or later.
